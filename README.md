## Overview

Мод для Stardew Valley, который отслеживает местоположение всех NPC на карте. 
Показывает маршруты перемещения персонажей и их текущее положение.

---

## Features

- 📍 Отображение всех NPC на карте
- 🗺️ Показ маршрутов перемещения
- 🔄 Интеграция с Content Patcher
- 🎨 Настраиваемые цвета маркеров
- ⚡ Автоматическое обновление позиций
